# SHOPPING_LISTS.md — Budget-Aware Recommendations

Framework for generating actionable shopping lists that respect budget, location, and host priorities.

## Three Budget Tiers

Each recommendation set is available in three tiers. Host chooses at analysis time.

### Tier 1: VALUE (Budget-Conscious)
- Focus: Essential items with maximum visual/functional impact
- Sourcing: Big-box retailers (IKEA, Amazon, local discount chains)
- Aesthetic: Minimal, clean, no-frills
- Average cost per item: $20–80 USD equivalent

**Example:**
- "Basic bedside table (IKEA TRYSIL or equivalent)" → $45
- "Simple white linen lampshade + warm bulb" → $30
- "Grey cotton throw blanket" → $25

### Tier 2: SIGNATURE (Mid-Range / Quality-Focused)
- Focus: Good design with durability and guest perception in mind
- Sourcing: Design retailers (Zara Home, H&M Home, Wayfair, mid-market local options)
- Aesthetic: Intentional, cohesive, slightly elevated
- Average cost per item: $80–200 USD equivalent

**Example:**
- "Solid wood bedside table with drawer (walnut or natural)" → $150
- "Ceramic or linen shade with brass fixture" → $90
- "Organic cotton throw blanket (premium weave)" → $80

### Tier 3: LUXURY (Premium / Statement)
- Focus: Hospitality-grade finishes and perceived value
- Sourcing: Premium brands (Restoration Hardware, Hay, Case Study, Schoolhouse Electric, high-end local)
- Aesthetic: Aspirational, memorable, Instagram-worthy
- Average cost per item: $200–600 USD equivalent

**Example:**
- "Solid walnut nightstand with brass hardware (Restoration Hardware or equivalent)" → $450
- "Ceramic or glass pendant with designer fixture" → $250
- "Premium linen throw blanket (Italian or Scandinavian)" → $180

## Item Categories & Quick Fixes

- Wayfair "Foundry Select" or equivalent: $150 each

## Shopping List Generation Logic

### Step 1: Analyze Gaps
For each Vitality dimension, identify missing items:
- Functional Anchors = 14/20 → Missing one bedside table
- Clutter = 8/20 → TV on console (wall-mount recommended)
- Lighting = 11/20 → No bedside lamps

### Step 2: Prioritize by Impact
Order items by ROI (revenue impact + ease of implementation):

1. **CRITICAL (blocks revenue):**
   - Missing bedside table/lamp pair
   - No entry storage
   - No workspace (urban studio)
   - Bathroom not photographed

2. **HIGH (affects guest satisfaction):**
   - Dark bedroom (no lamps)
   - Sofa with nowhere for drinks
   - TV on large console (floor space issue)
   - Clashing colours

3. **MEDIUM (nice-to-have):**
   - Additional decor (pillows, art, plants)
   - Upgraded textiles
   - Lighting upgrades beyond basics

### Step 3: Budget Allocation
Host sets budget (e.g., $1500 RM):

1. **60% to CRITICAL items** ($900)
   - Bedside table + lamp pair: $400
   - Entry hooks + shoe rack: $300
   - Workspace desk + chair: $200

2. **30% to HIGH items** ($450)
   - Wall-mount TV service: $300
   - Additional lighting: $150

3. **10% to MEDIUM items** ($150)
   - Throw pillows, art, plants

### Step 4: Regional Sourcing
Pull vendors and links based on location:
- "Bedside table" + Malaysia → IKEA Malaysia, Mr DIY, Amazon.com.my
- "Bedside lamps" + Thailand → IKEA Thailand, HomePro, Emporium shopping center
- "Entry hooks" + Spain → IKEA Spain, Zara Home, local artisans

## "Why" Justifications (For Each Item)

Every shopping list item must include a plain-language "why":

### ✅ Good Examples:
- "Paired bedside lamps create a hotel-standard sleep environment and tell guests this bedroom was thoughtfully designed."
- "A sofa side table lets guests set drinks, phones, and remotes without reaching across the room — improves comfort and perceived quality."
- "Wall-mounting the TV reclaims floor space in the living area, making the room feel 20% larger and less cramped."
- "Entry hooks solve the #1 guest pain point: where do I hang my jacket? Hooks = guest trust."

### ❌ Bad Examples:
- "You need a bedside lamp."
- "Wall-mount the TV."
- "Add hooks."

**Rule:** Explain the **guest experience benefit**, not the technical requirement.

## Shopping Preference Options

Host chooses how they want vendors listed:

### 1. ONLINE ONLY
Links to: Amazon, Wayfair, Zara Home, IKEA, H&M Home, etc.
**Use case:** Host has time to wait for shipping, prefers flat-pack convenience, wants comparison shopping.

### 2. IN-STORE ONLY
Mentions local retailers in the area (department stores, design shops, vintage markets).
**Use case:** Host wants immediate gratification, wants to see/touch items before buying, prefers local economy.

### 3. BOTH (default)
Primary link (e.g., IKEA) + alternative (e.g., "or try Mr DIY for local option").
**Use case:** Host wants maximum flexibility.

### 4. CURATED VENDOR ONLY
Matches items exclusively to pre-approved vendors (IKEA, Amazon, Wayfair, etc.).
**Use case:** Host wants simplicity, doesn't want to research.

## Budget Constraint Logic

If shopping list exceeds budget, app prioritizes:

1. **CRITICAL items** (bedside tables, entry storage, workspace)
2. **MUST-HAVE anchors** (sofa side table, lamps)
3. **NICE-TO-HAVE decor** (throws, pillows, art)

If still over budget, app suggests:
- Value tier instead of Signature tier
- "Show It" items instead of buying new
- Phased approach: "Month 1: bedside tables ($400). Month 2: entry storage ($250)."

## Philosophy

> **"Every dollar in the shopping list should increase revenue, guest satisfaction, or both."**

- Cheap doesn't mean low-impact. A $30 hook solves a $500/month revenue leak (bad reviews, cancellations).
- Expensive doesn't guarantee ROI. A $1000 vintage art piece is wasted if the bed has no lamps.
- Regional matters. An IKEA table in Malaysia costs 30% less than equivalent US pricing. Recommendations adapt.
- Honesty over sales. If a property is clean and functional, don't recommend "upgrades" it doesn't need.
