## THE PROBLEM: DESIGNER SPEAK FAILS HOSTS

### Current Output (❌ Wrong)
```
"The bedroom lacks visual interest and coherence. Consider adding artwork
to create a focal point, and incorporate textiles to establish a softer 
aesthetic that elevates the overall spatial narrative."
```

**Why this fails:**
- Host thinks: "Nice-to-have, optional, expensive"
- Actually means: "This room is losing bookings because it feels cold"
- Result: Host doesn't prioritize the fix → continues to lose $500/month in revenue

### Target Output (✅ Correct)
```
"Bedroom missing bedside lamps → 15% fewer bookings from couples seeking 
'cozy' experiences → -$180/month revenue impact. Fix: 2x $25 lamps = $50 
investment, expected ROI +$180/month = 3.6x payback in month 1."
```

**Why this works:**
- Host thinks: "$50 vs. -$180/month? I'm doing this now."
- Clear causation: Missing item → Specific filter lost → Quantified revenue impact
- Result: Host prioritizes fix immediately → Booking uplift visible in 2-3 weeks

## THE HIERARCHY: REVENUE-FIRST DECISION TREE

When analyzing a property, prioritize recommendations using this hierarchy:

### TIER 1: AIRBNB FILTER TRIGGERS (Highest ROI)
**These directly impact booking conversion via Airbnb search filters**

```
FILTER: "Dedicated Workspace"
- Missing: Desk + chair + surface
- Revenue impact: -15% bookings (remote workers filtered out)
- Fix cost: $100-300
- Monthly upside: +$200-400
- Priority: CRITICAL (P1)
- Host speak: "Remote workers are 22% of bookings. Lose this filter = lose $200/month."

FILTER: "King Bed"
- Missing: Actual king bed (not 2 singles)
- Revenue impact: -12% bookings (couples avoiding)
- Fix cost: $400-800
- Monthly upside: +$150-250
- Priority: CRITICAL (P1)
- Host speak: "Couples book 12% of nights at +20% premium. Upgrade to real king = +$150/month."

FILTER: "Baby Gear"
- Missing: Crib, high chair, baby-proofing
- Revenue impact: -8% bookings (families with infants filtered out)
- Fix cost: $100-300
- Monthly upside: +$80-150
- Priority: CRITICAL (P1) if family property
- Host speak: "Family bookings +30% premium. Add crib = +$100/month, $200 investment."

FILTER: "Essentials"
- Missing: WiFi, kitchen basics, cleaning supplies
- Revenue impact: -18% bookings (instant deal-killer)
- Fix cost: $50-150 (mostly free items already owned)
- Monthly upside: +$200-300
- Priority: CRITICAL (P1)
- Host speak: "A guest can't book without this. Non-negotiable."
```

### TIER 2: REVIEW PROTECTION (Booking Conversion via Trust)
**These prevent negative reviews that tank future bookings**

```
PILLAR: "Cleanliness & Condition"
- Missing: Visible wear, stains, odors, broken items
- Revenue impact: -25% repeat bookings (if reviews say "dirty")
- Fix cost: $0-100 (cleaning, repairs)
- Monthly upside: +$400+ (reputation recovery)
- Priority: CRITICAL (P1)
- Host speak: "A 3.5-star review kills next 20 bookings. Replace sheets, deep clean."

PILLAR: "Functionality Integrity"
- Missing: Working appliances, shower, locks, heating
- Revenue impact: -30% bookings (broken property = cancellations)
- Fix cost: $100-1000+
- Monthly upside: +$300-500
- Priority: CRITICAL (P1)
- Host speak: "A broken shower = instant refund + cancellation. Fix it now."

PILLAR: "Guest Comfort (Soft Factors)"
- Missing: Comfortable bedding, warm blankets, adequate seating
- Revenue impact: -8% bookings (reviews say "uncomfortable")
- Fix cost: $100-300
- Monthly upside: +$100-200
- Priority: HIGH (P2)
- Host speak: "Better bedding = 5-star reviews = repeat bookings = +$100/month."
```

### TIER 3: LISTING OPTIMIZATION (Nice-to-Have, Lower ROI)
**These improve photos but have lower booking impact**

```
PILLAR: "Visual Staging & Aesthetics"
- Missing: Artwork, color coordination, decorative pillows
- Revenue impact: -3-5% bookings (subtle perception difference)
- Fix cost: $50-200
- Monthly upside: +$50-100
- Priority: LOW (P3, do after P1/P2)
- Host speak: "Nicer photos get +3% click-throughs, but workspaces matter more."
```

## HOST SPEAK CONSTRAINT: ENFORCE BUSINESS LANGUAGE

### Before (❌ Designer Speak)
```
"The living room would benefit from layered lighting with warm color 
temperatures to establish an inviting, residential aesthetic."
```

### After (✅ Host Speak)
```
"Living room only has overhead lights → harsh, institutional feel → reviews 
say 'cold/unwelcoming' → -5% repeat bookings. Add: 1-2 table lamps ($40-60) 
+ warm bulbs ($10). Expected ROI: +$50-100/month from improved reviews."
```

### Translation Rules

| Designer Phrase | Host Translation | ROI Metric |
|---|---|---|
| "Lacks visual interest" | "Missing focal point photos lose clicks" | -3% CTR → -$50/month |
| "Weak spatial flow" | "Furniture layout blocks doorways/movement" | -1 guest star rating → -$100/month |
| "Institutional aesthetics" | "Feels corporate/temporary, not 'home'" | -8% family bookings → -$150/month |
| "Color incoherence" | "Mismatched styles look cheap in photos" | -5% click-to-inquiry → -$70/month |
| "Inadequate layering" | "Single overhead light = photos look harsh" | -4% listing rank → -$80/month |
| "Staging deficit" | "Photos don't show comfortable living space" | -6% conversion → -$120/month |

## CONTEXTUAL AWARENESS: LISTING-TYPE WEIGHTING

The same fix has different ROI impact depending on listing type. Inject this logic:

### LISTING TYPE: URBAN STUDIO (1-2 guests, business travelers)

```
HIGHEST ROI FIXES:
1. Workspace (P1) - Remote workers are 40% of urban studio bookings
   → Missing desk = -$300/month
2. WiFi + fast connectivity (P1) - Non-negotiable for workers
   → Missing = -$400/month
3. Comfortable chair + sofa (P2) - Long-stay comfort
   → Poor seating = -$150/month

LOWEST ROI:
- Artwork & decoration (P3) - Urban travelers don't care about aesthetics
- Large living furniture (not applicable)
- Family amenities (crib, high chair)
```

### LISTING TYPE: RESORT (4-6 guests, families, vacationers)

```
HIGHEST ROI FIXES:
1. Comfortable beds (P1) - Families sleep deeply after day trips
   → Poor bedding = -$200/month (negative sleep reviews)
2. Spacious seating (P1) - Family bonding happens on sofa
   → Undersized furniture = -$250/month (cramped reviews)
3. Shared bathroom organization (P2) - 6 people, 2 bathrooms = chaos
   → Poor organization = -$150/month (bathroom reviews tank)

LOWEST ROI:
- Workspace (families don't care)
- WiFi quality (lower priority vs. leisure travelers)
- Individual lighting (families care less about ambience)
```

### LISTING TYPE: CITY APARTMENT (2-4 guests, couples + small families)

```
HIGHEST ROI FIXES:
1. King bed or premium bedding (P1) - Couples book 8% premium
   → Poor bed = -$180/month
2. Workspace for 1-2 people (P1) - Hybrid work culture
   → Missing desk = -$200/month
3. Living room seating quality (P1) - Couples spend evenings together
   → Poor sofa = -$100/month

MEDIUM ROI:
- Bathroom lighting (couples care about aesthetics)
- Kitchen functionality (small families cook)
```
