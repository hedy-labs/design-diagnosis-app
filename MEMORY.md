## The Airbnb Visual Diagnostic App (Project)
**Concept:** AI concierge for Airbnb/VRBO hosts. Upload listing photos or paste link → get a Vitality Score, room-by-room diagnostic, and a budget-aware shopping list. Replaces real-life stager/decorator consultation.

**Target user:** Short-term furnished rental owners (Airbnb, VRBO, etc.)
**Core value:** Instant, affordable alternative to human staging consultation ($29–49 vs $500–2000)

**Four diagnostic pillars:**
1. Visual Recognition Logic (photo analysis)
2. Fix/Solution Database ("Stager's Logic" knowledge graph)
3. Budget-Integrated Shopping API (3 tiers: Value / Signature / Luxury)
4. User UI/UX (Vitality Score dashboard + one-tap fulfillment)

**Stager's Logic Framework (EXPERIENCE_LOGIC.md):**
- Entryway Friction: hooks, shoe rack, bench at entry — flagged as high-priority UX failure
- Functional Anchors: paired bedside tables/lamps, sofa side tables — "Incomplete Sleep Sanctuary"
- Workspace Utility: dedicated desk + chair — essential for modern guests
- Shower Utility: shelf, caddy, or bench — "Shower Utility Deficit"
- Proportional Harmony: table/furniture scale vs. guest count
- Integrity Check: staging props (wine, chocolate, food) not provided = "Staging Drift" flag
- Color Coherence, Lighting Temperature, Clutter Density

**Hidden Friction Protocol (non-visual):**
Linen quality (no pilling/stains), top sheet, 2 pillows per guest (firm+soft), 2 towels/guest/week, 3 facecloths/guest/week, potable water where needed, 3 TP rolls/week, kitchen essentials (can opener, scissors, sharp knives, peeler, etc.), bathroom kit (plunger, drain catcher, squeegee, hooks), bedroom (mattress protector, pillow protectors, hangers), welcome basket (2 drinks, salty+sweet snack), power bars, laundry supplies, 2 sets of keys.

**Pricing models discussed:**
1. Pay-per-report: $29–$49/listing audit
2. Property Manager subscription: $49–$99/month (5+ properties)
3. Affiliate play: free/cheap reports, revenue from shopping list affiliate links (Amazon, Wayfair, etc.)

**Competitors:**
- Traditional stagers/Havenly/Modsy → slow, expensive, not STR-specific
- Virtual Staging AI (BoxBrownie) → fake furniture for real estate, no physical shopping list
- Listing Optimizers (PriceLabs, Rankbreeze) → price/text optimization, no decor intelligence
- Our edge: only tool judging actual interior design quality + actionable shopping list

## Backend Implementation: Polite-Pacing & Error Handling (2026-03-30)

**Problem Solved:** Rate limit loop + crashes. Solution: 5-second delay + local queue + "System Cooling Down" message.

**Four New Files Created:**

1. **api_manager.py** (13.3 KB)
   - APIManager class: enforces 5s polite-pacing between all API calls
   - RequestQueue: persists failed requests to disk (survives app restart)
   - Exponential backoff: 5s → 10s → 20s → 40s on retries
   - Error states: "pending" | "cooling_down" | "success" | "failed"
   - Local persistence: api_request_queue.json on disk

   - analyse_listing_managed(): drop-in replacement for analyse_listing()
